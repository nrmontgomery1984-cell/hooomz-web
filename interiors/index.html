How We Work

Six stages. Every job.
No exceptions.

SCRIPT is the production method we run on every renovation — so nothing gets skipped, no surprises get buried, and turnover is clean.

S
Step 01
Shield

Surface protection placed, existing finishes documented. Nothing moves without substrate sign-off.

C
Step 02
Clear

Demo and removal. Substrate issues flagged immediately — no surprises buried under new floors.

R
Step 03

Layout confirmed, materials staged and verified before installation begins.

I

Daily progress logged, photos documented by room. You can see it happen in real time.

P

T
